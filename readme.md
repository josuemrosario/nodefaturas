# Sistema Faturas

nodefaturas é um sistema desenvolvido em aula para aprender a usar o node com o Express 

## Tecnologias Utilizadas
- node
- expressJS
- MongoDB

## Instalação

Para usar instale as dependencias, compile o CSS e suba o servidor

```sh
npm i
npm run devcss
npm run dev
```
